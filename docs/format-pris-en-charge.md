# Format de fichier pris en charge

Formats pris en charge parmi les plus utilisé : OpenDocument (ods, odt, etc.), PDF, HTML, Microsoft Office (DOC/DOCX/RTF, XLS/XLSX, PPT/PPTX)

# Format médias

- Image : JPEG, PNG, GIF, BMP et SVG
- Média : MP4, MP3, WAV, OGG et WEBM
- Text : TXT, XML, JS, HTML, JSON et TS


## Format texte

Dans le détail, voici les formats pris en charge :
